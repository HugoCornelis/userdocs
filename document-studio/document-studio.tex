\section*{The Studio}

The Studio has a GUI front-end to the \href{../document-model-container/document-model-container.pdf}{Model Container} and allows browsing and visualization of a model. Note that the Studio is not a graphical editor or construction kit. External applications such as \href{http:\\www.physiol.ucl.ac.uk/research/silver_a/neuroConstruct/}{neuroConstruct} should be employed for these purposes.

Additionally, the Studio comes with a shell command that uses the Model Container \href{http://www.swig.org/}{Swig} bindings to access the model stored by the Model Container.

\subsubsection*{Features}

\begin{itemize}

\item Visualizes models stored by the Model Container.

\item  Allows a model to be queried from the \href {../introduction-gshell/introduction-gshell.pdf}{GENESIS shell} command line.

\end{itemize}
The Studio is a necessary part of the \href{../document-project-browser/document-project-browser}{Project Browser}.
